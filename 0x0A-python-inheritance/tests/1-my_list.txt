=================
Module: 1-my_list
=================

``MyList(list)`` is a class that inherits from `list` and adds
a `print_sorted()` method.

Import
======

    >>> MyList = __import__('1-my_list').MyList

Types
=====


::

	>>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]
